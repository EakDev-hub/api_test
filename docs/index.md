# DateTime & Timezone API with Calculator

Welcome to the DateTime & Timezone API documentation. This service provides REST endpoints for datetime operations, timezone conversions, and mathematical calculations.

## Overview

This API offers:

- **DateTime Operations**: Get current datetime, timezone-specific datetime, and convert between timezones
- **Timezone Management**: List all available timezones
- **Calculator**: Perform mathematical operations including addition, subtraction, multiplication, division, power, square root, and percentage calculations

## Quick Start

### Installation

```bash
npm install
```

### Running the Server

```bash
npm start
```

The server will start on `http://localhost:3005`

### Development Mode

```bash
npm run dev
```

## API Documentation

Interactive Swagger documentation is available at:
```
http://localhost:3005/api-docs
```

## Architecture

This service is built with:
- **Node.js** - Runtime environment
- **Express** - Web framework
- **Moment-timezone** - Timezone handling
- **Swagger** - API documentation
